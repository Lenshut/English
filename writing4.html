Writing • Level 4

Intermediate Writing

Construct robust, multi-clause paragraphs. Practice formal emails, embedded questions, relative clauses, and logical opinion structures.

💡 10 Writing Topics

Choose a topic below and write a detailed paragraph (8-10 sentences) using Level 4 structures.

1. A Recent Trend: Describe a trend (e.g., remote work) and use relative clauses (who/which) to explain it.
2. Work vs Home: Compare working in an office vs home using 'Furthermore' and 'However'.
3. A Wild Dream: Write about what you would do if you were invisible (2nd Conditional).
4. A Past Regret: Use the 3rd Conditional to explain a mistake you made and how you could have avoided it.
5. Professional Request: Write an email asking a colleague for help using embedded questions (Do you know...).
6. The Best Option: Compare 3 products and declare one 'the best', giving 2 reasons.
7. Detailed Complaint: You bought a bad phone. Explain what is wrong using 'what's worse' and 'keeps crashing'.
8. Technology Pros/Cons: Discuss a gadget. 'On one hand it is useful, but on the other hand...'
9. Meeting Notes: Summarize a fake meeting using 'First of all', 'Secondly', and 'To sum up'.
10. Phrasal Verb Story: Write a short story using at least 4 business phrasal verbs (figure out, wrap up, look into).

📧 Formal vs Informal Email

Practice shifting your register by writing two emails on the same topic.

Task 1: Formal Request

Write an email to a vendor (Mr. Smith) asking for an update on an order. Use embedded questions (Could you tell me...) and polite requests (I would appreciate it if...).

To: vendor@supply.com
Subject: Order Status Inquiry

Task 2: Informal Request

Write to a close coworker (Sarah) asking her to send you the same order file. Use casual language and phrasal verbs.

To: sarah@company.com
Subject: Quick favor!

⚖️ Structured Opinion

Write a cohesive, 4-sentence argumentative paragraph using specific logical connectors.

Topic: Should people work a 4-day work week instead of 5 days?

  • Sentence 1: State your opinion clearly (In my opinion...).
  • Sentence 2: Give the main reason.
  • Sentence 3: Add a secondary benefit using "Furthermore".
  • Sentence 4: Acknowledge a downside using "While it is true that..." or "However".
