Upper-Intermediate Exercises

Test your knowledge of Upper-Intermediate concepts: Perfect Continuous Tenses, Past Perfect, Cleft Sentences, Debate Logic, and Register Adaptation.

✍️ Fill in the Blanks

1. I have ______ studying English for three hours.
2. By the time I arrived, the train had already ______.
3. My boss, ______ has worked here for 20 years, is retiring.
4. ______ the first option is cheaper, the second provides better value. (Whereas)
5. ______ reviewed the data, the committee approved the project. (Having)
6. What I really ______ is a vacation.
7. We must leave now; ______, we will miss the flight.
8. Just because he is rich, that doesn't ______ mean he is happy.
9. Let me play devil's ______ for a second.
10. We need to figure ______ a plan.

📝 Short Answer

1. Write a sentence using Present Perfect Continuous.

2. Combine using Past Perfect: I ate. You arrived.

3. Create a non-defining relative clause.

4. Write a sentence using "Whereas".

5. Emphasize using a cleft sentence: I need coffee.

6. Give a condition using "Provided that".

7. Dismantle logic using "doesn't necessarily mean".

8. Translate "fix the problem" into formal English.

9. Identify a "root cause" of traffic.

10. Write a 3rd conditional sentence.

🗣️ Speaking Tasks

Practice these aloud with a partner or teacher.

01

Complain about a task using Present Perfect Continuous.

02

Tell a story placing events in order with Past Perfect.

03

Introduce a colleague adding extra info with a non-defining clause.

04

Compare two options in a business pitch using "Whereas".

05

Emphasize a core issue using a "What..." cleft sentence.

06

Issue a strict condition using "Provided that".

07

Dismantle a fallacy: "Just because... doesn't necessarily mean...".

08

Deflect an argument with "That's beside the point".

09

Switch register: Ask a friend for help, then ask a boss formally.

10

De-escalate an angry customer using "I understand your frustration".
